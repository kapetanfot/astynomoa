# astynomoa - Αστυνομόα
Σύνθετο από τις λέξεις '''αστυνομία''' + '''-μόα''' 
=={{ε}}==
{{νεολογισμός|el}}

=== Ετυμολογία ===
Σύνθετο από τις λέξεις '''αστυνομία''' + το σατιρικό επίθημα '''-μόα'''. Το επίθημα '''-μόα''' προέρχεται από παραμόρφωση της λέξης «[[μαλακία]]» και χρησιμοποιείται για να δηλώσει ανικανότητα ή γκάφα.
